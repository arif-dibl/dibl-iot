# DIBL IoT Smart Controller - Product Features

**Device Type:** Environmental Monitoring and Control System  
**Application:** Agriculture, Greenhouse, Indoor Farming

---

## What This Device Does

The DIBL IoT Smart Controller monitors your growing environment and automatically controls equipment like pumps, lights, and fans based on schedules you set. It connects to the cloud so you can view data and control your system from anywhere.

---

## Key Features

### 1. Environmental Monitoring

The device continuously measures conditions in your growing area.

| What It Measures | Description |
|------------------|-------------|
| Temperature | Air temperature in Celsius or Fahrenheit |
| Humidity | Air moisture percentage |
| Soil Moisture | How wet or dry your soil is (up to 4 zones) |
| Light Level | Brightness as a percentage |

**Advanced Soil Analysis (Optional NPK Sensor)**

| Measurement | What It Tells You |
|-------------|-------------------|
| Soil Temperature | Ground temperature |
| Soil Moisture | Water content in soil |
| pH Level | Soil acidity or alkalinity |
| Nitrogen (N) | Key nutrient for leaf growth |
| Phosphorus (P) | Key nutrient for root and flower development |
| Potassium (K) | Key nutrient for overall plant health |
| Electrical Conductivity | Total nutrients dissolved in soil |

---

### 2. Equipment Control

Control up to 4 pieces of equipment (relays) directly from the device or remotely.

| Output | Typical Use |
|--------|-------------|
| Output 1 | Water pump, irrigation valve |
| Output 2 | Grow lights |
| Output 3 | Exhaust fan, ventilation |
| Output 4 | Heater, humidifier, or other equipment |

**Control Methods:**
- Press buttons on the device
- Use the cloud dashboard from your phone or computer
- Set automatic schedules (see Timers below)

---

### 3. Automatic Scheduling (12 Timers)

Set up to 12 independent timers to automate your equipment. Each timer can:

| Setting | Options |
|---------|---------|
| Turn On Time | Any hour and minute (e.g., 6:30 AM) |
| Turn Off Time | Any hour and minute (e.g., 6:00 PM) |
| Active Days | Select specific days or every day |
| Controlled Outputs | Choose one or more outputs to control |
| Enable/Disable | Turn the timer on or off without deleting it |

**Scheduling Examples:**

| Timer Purpose | On Time | Off Time | Days | Outputs |
|---------------|---------|----------|------|---------|
| Morning irrigation | 6:30 AM | 7:00 AM | Every day | Output 1 |
| Grow lights | 6:00 AM | 8:00 PM | Weekdays | Output 2 |
| Night ventilation | 10:00 PM | 6:00 AM | Every day | Output 3 |
| Weekend heating | 7:00 AM | 9:00 PM | Sat, Sun | Output 4 |

---

### 4. Cloud Connectivity

The device connects to your Wi-Fi network and communicates with the cloud server.

| Feature | Description |
|---------|-------------|
| Remote Monitoring | View all sensor readings from anywhere |
| Remote Control | Turn outputs on or off from your phone |
| Timer Sync | Update schedules from the cloud dashboard |
| Secure Connection | Encrypted communication protects your data |
| Auto-Reconnect | Device automatically reconnects if connection drops |

---

### 5. Built-in Display

A small screen on the device shows real-time information.

| Screen | What It Shows |
|--------|---------------|
| Environment Dashboard | Temperature, humidity, moisture, connection status |
| NPK Dashboard | Soil nutrient levels and pH |
| Light Meter | Current light level with visual bar |
| Time and Date | Current time and day of week |
| Connection Status | Shows if device is connected to cloud |

**Navigation:**
- Turn the knob to switch between screens
- Press the knob to change settings or toggle temperature units

---

### 6. Status Indicators

Three LED lights show the device status at a glance.

| LED | Meaning |
|-----|---------|
| Timer LED | Solid light when any timer is active |
| Server LED | Blinking when connected to cloud |
| Wi-Fi LED | Blinking pattern shows network status |

---

### 7. Easy Setup

| Setup Feature | Description |
|---------------|-------------|
| Wi-Fi Configuration | Connect to any Wi-Fi network using your phone |
| Automatic Registration | Device registers itself with the cloud automatically |
| Sensor Calibration | Fine-tune sensors through the on-device menu |
| Manual Time Setting | Set time manually if no internet is available |

**First-Time Setup:**
1. Power on the device
2. Connect your phone to the device's Wi-Fi (DIBL IoT)
3. Enter your home Wi-Fi details
4. Device connects to cloud automatically

---

### 8. Firmware Updates

The device can receive software updates over Wi-Fi without any physical connection needed.

## Sensor Zones

The device supports multiple sensor zones for larger growing areas.

| Zone | Sensors Available |
|------|-------------------|
| Onboard | Temperature, humidity, light |
| Zone 1-4 | Temperature, humidity, soil moisture |
| NPK Zone 1-4 | Full soil analysis (7 parameters each) |

## Supported Equipment Types

The 4 relay outputs can control:

| Equipment Type | Notes |
|----------------|-------|
| Water pumps | Up to 10A load per output |
| Solenoid valves | For irrigation control |
| Grow lights | LED or fluorescent |
| Fans and blowers | Ventilation control |
| Heaters | Space or soil heaters |
| Humidifiers | Misting systems |
| Any on/off device | If it can be switched, this device can control it |

---

## Data You Can Track

All readings are sent to the cloud where you can:

| Capability | Description |
|------------|-------------|
| View Current Values | See real-time sensor readings |
| Check Output Status | See which outputs are on or off |
| Review Timer Settings | View and update all 12 timers |
| Historical Data | Access past readings through the dashboard |

---

## Offline Operation

If the internet connection is lost:

| Feature | Offline Behavior |
|---------|------------------|
| Timers | Continue running on schedule |
| Local Buttons | Still control outputs manually |
| Display | Shows all sensor readings |
| Automatic Reconnect | Device reconnects when internet returns |
